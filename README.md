## 简介
科学膜蛤，学习一个，提高姿势水平；
谈笑风生，一派胡言，祝你身体健康。

## 功能
【说】：语录、采访、访问对话、诗词等作品。
【学】：考据、同人、段子、知乎精选等知识。
【逗】：高清、原生、颜艺、鬼畜、成语、恶搞、暴走漫画等表情，支持动图。
【唱】：古典、流行、动画等音乐。
